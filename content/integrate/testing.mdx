## Smoke test

```sh
curl -sX POST https://light.zcash.me/zns-testnet \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"status","params":{}}'
```

## Wallet checklist

- `new ZNS().resolveName("alice")` returns a result with `address`, `txid`, `height`, `nonce`, `last_action`
- `new ZNS().resolveName("definitelynotregistered")` returns `null`, not an error
- `new ZNS().resolveAddress(unifiedAddress)` returns an array (possibly empty)
- `zns.isValidName("Alice")`, `zns.isValidName("my-name")`, `zns.isValidName("a".repeat(70))` all return `false`
- Wrong URL throws an Error (e.g., closed port, 404, etc.)
- `await zns.verify()` throws if UIVK doesn't match expected (call after constructing ZNS)

## Explorer checklist

- `events` with `since_height` streams correctly (filter is strict `>`, not `>=`)
- All seven action verbs render: `CLAIM UPDATE LIST DELIST BUY RELEASE SETPRICE`
- `SETPRICE` events have an empty `name` field
- `resolveAddress(address)` returns an array; empty array is the normal no-names case
