# First Steps

## Claims

Claiming a new name is currently a web flow at [zcashnames.com](https://www.zcashnames.com). Wallets that want to support claims today should deep-link or open the web flow.

## What to read

- [Wallet: Resolve a Name](/docs/integrate/wallet-resolution) for the implementation path
- [Explorer Integration](/docs/integrate/explorer-integration) if you want registry activity in an explorer or app
- [Testing](/docs/integrate/testing) for endpoints and fixtures

## Current rollout

- mainnet is live in beta
- wallet resolution has not shipped yet
- Edge and Unstoppable are implementing support
## SDKs and spec

There are first-party SDKs for TypeScript, React Native, Rust, Python, Kotlin, Swift, Go, and Dart. See [SDKs](/docs/sdk).

The machine-readable RPC spec is at [`openrpc.json`](https://github.com/zcashme/ZNS/blob/master/openrpc.json).
