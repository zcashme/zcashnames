# Libraries

The TypeScript SDK wraps the JSON-RPC API and memo protocol. If your language isn't TypeScript/JavaScript, hit the indexer directly via [Direct RPC](/docs/sdk/direct-rpc), or generate a typed client from [`openrpc.json`](https://github.com/zcashme/ZNS/blob/master/openrpc.json).

The SDK lives in [`zcashme/ZNS/sdk/typescript`](https://github.com/zcashme/ZNS/tree/master/sdk/typescript).

## TypeScript / JavaScript

```sh
npm install zcashname-sdk
```

```typescript
import { ZNS } from "zcashname-sdk";

const zns = new ZNS();
const reg = await zns.resolveName("alice");
```

Reference: [TypeScript SDK](/docs/sdk/typescript).

## React Native

```sh
npm install zcashname-sdk-react-native zcashname-sdk
```

```tsx
import { ZNSProvider, useZNS } from "zcashname-sdk-react-native";

// Wrap your app in <ZNSProvider />, then use hooks
function MyComponent() {
  const { resolveName, isAvailable } = useZNS();
  // ...
}
```

Wraps `zcashname-sdk` with a context provider and data-fetching hooks.

## No SDK for your language?

Hit the indexer directly. The API is JSON-RPC 2.0 over HTTP - any language with an HTTP client and a JSON parser can talk to it in under 20 lines. See [Direct RPC](/docs/sdk/direct-rpc) for `curl`, Rust, Python, and Go examples.

The canonical machine-readable spec is at [`openrpc.json`](https://github.com/zcashme/ZNS/blob/master/openrpc.json). Point any [OpenRPC code generator](https://github.com/open-rpc/generator) at it to scaffold a typed client.

PRs adding ZNS helpers to existing Zcash libraries are welcome. Open an issue at [`zcashme/ZNS`](https://github.com/zcashme/ZNS/issues) and we'll help with review and test vectors.
