# Security Reviewer / Protocol Security Consultant

## Purpose

Provide independent review before production. Improve trust in protocol, SDK, resolver, signing, and wallet-facing flows, and catch design or implementation risks before launch.

## Responsibilities

- Review protocol assumptions and trust model.
- Review resolver behavior and client verification assumptions.
- Review SDK behavior and wallet-facing flows.
- Review signing, ownership, update, and authorization flows.
- Maintain or create a threat model.
- Produce concise findings with severity, exploit path, and remediation guidance.

## Required

- Application security or protocol security experience.
- Ability to review code and architecture.
- Strong threat-modeling skills.
- Clear, brief communication about practical risk.
- Independent technical judgment.

## Strong Plus

- Zcash, privacy, wallet, naming, or identity-system experience.
- Experience with replay protection, authorization, and key management.
- Prior public audit reports or security research.

## Success

- Production launch receives a meaningful security review before release.
- Findings are actionable.
- Fixes are validated or unresolved items are clearly called out.
