# What is ZcashNames?

ZcashNames lets someone publish a human-readable name like `alice.zcash` that resolves to a Zcash unified address.

The registry is derived from standard Zcash shielded transactions, not from a centralized off-chain database.

## What problem it solves

A Zcash unified address looks like so.

```
u1gphl7vrklduuv96kpw4eetx4vrs8nnk7w9tuzvppyuuctw0tuskkpmfulrjapr05zh78p3chpxhx3tm28qau3uwd36k94vgucpxphyv5hkg36nhvr4ax
```

With ZcashNames, a wallet or app can resolve `alice.zcash` to a user's address before sending.

## Why the on-chain part matters

ZNS uses the Zcash blockchain to serve name registration.

- Name registrations are anchored in confirmed Zcash transactions
- Registry state can be re-created by anyone running the ZNS indexer


## Already using zcash.me?

If you already like having a profile on `zcash.me`, ZcashNames is the upgrade path for that identity.

Registering a ZcashNames name automatically creates the matching `zcash.me` profile - you may customize your profile with a bio, links, and an avatar.

A .zcash name also becomes an on-chain identifier that Zcash apps (wallets, block-explorers, forums etc. ) can resolve.
