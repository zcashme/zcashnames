# What is ZcashNames?

A Zcash unified address looks like this:

```
u1gphl7vrklduuv96kpw4eetx4vrs8nnk7w9tuzvppyuuctw0tuskkpmfulrjapr05zh78p3chpxhx3tm28qau3uwd36k94vgucpxphyv5hkg36nhvr4ax
```

ZcashNames lets people pay you at `alice.zcash` instead. The name is short, memorable, and resolves to your unified address every time.

## Where it stands today

Testnet is active with registrations open and a stable protocol. Mainnet is in beta — the indexer is running, but treat it as experimental software.

[Cake Wallet](https://cakewallet.com) ships name resolution as of v5.8.0. Other wallet integrations are in progress. For a full picture of what is deployed and who is using it, see [Ecosystem](/docs/ecosystem).

The canonical spec is the [`openrpc.json`](https://github.com/zcashme/ZNS/blob/master/openrpc.json) file plus the [Memo Format](/docs/protocol/memo-format) page.
