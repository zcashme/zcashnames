# JSON-RPC Reference

The ZNS indexer exposes a read-only JSON-RPC 2.0 API with four methods: `resolve`, `listings`, `status`, and `events`.

The canonical spec is [`openrpc.json`](https://github.com/zcashme/ZNS/blob/master/openrpc.json). Use the explorer below to hit a live indexer, or point any [OpenRPC code generator](https://github.com/open-rpc/generator) at it to scaffold a typed client.

For curl and language examples without the SDK, see [Direct RPC](/docs/sdk/direct-rpc). For signature verification, see [Signature Scheme](/docs/protocol/signatures).

|---------|-----|
