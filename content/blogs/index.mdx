# ZcashNames Blogs

Three Git-backed series live here so contributors can open issues, suggest edits, and publish Markdown directly from the repo.

## Series

### [Updates](/blogs/updates)

Core product notes, roadmap movement, and release communication.

### [Launch](/blogs/launch)

Pricing, rollout context, launch checklists, and release-phase notes.

### [Builders](/blogs/builders)

Integration notes, partner implementation details, and developer-facing stories.

## How to contribute

All blog pages live under `content/blogs/` in this repository.

- Open an issue for structural changes or missing topics.
- Edit an existing `.mdx` file for copy changes.
- Add a new `.mdx` post file inside one of the series folders for a new post.
