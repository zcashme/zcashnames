# ZcashNames

ZcashNames maps a human-readable name like `alice.zcash` to a Zcash unified address. The protocol is called **ZNS** and stores registrations in standard Zcash shielded transactions, not in a private company database.

If you already use `zcash.me`, this is the on-chain version of that identity. Registering a name creates a matching `zcash.me` profile automatically, and you can add your bio, links, and avatar later.

Mainnet is live in beta. During beta, claim prices are set to **1/100th** of the intended production tiers.
